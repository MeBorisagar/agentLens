# AgentLens

Open-source observability and replay platform for AI agents.

## Vision

AI agents fail silently.

AgentLens helps developers:
- trace executions,
- inspect tool calls,
- replay failures,
- analyze latency and cost,
- debug agent behavior.

---

# MVP Features

- Trace Capture SDK
- Replay System
- Timeline Visualization
- Tool Inspection
- Cost & Latency Metrics
